# A股 AI 选股模型

一个本地运行的 A 股 AI 选股 Demo，使用 `FastAPI + LightGBM + 原生前端网页` 构建，支持在本地完成真实数据准备、特征工程、模型训练、股票打分和结果展示。

当前版本已经从“样例演示模式”升级到“真实 A 股历史数据优先模式”：

`AKShare历史数据 -> 特征生成 -> 模型训练 -> 选股打分 -> 网页展示`

如果真实数据获取失败，系统仍然可以自动退回到本地样例数据，保证网页和训练流程不中断。

## 功能特性

- 本地网页控制台，直接在浏览器查看选股结果
- FastAPI 后端接口，提供总览、候选股、重新训练和刷新真实数据能力
- 默认使用 `AKShare` 拉取真实 A 股历史日线数据
- 使用 `LightGBM` 做基础选股打分模型
- 支持一键重新训练模型
- 支持一键刷新真实数据并重新出候选股
- 完全本地运行，不依赖云服务器

## 当前真实数据方案

当前版本为了提高稳定性，采用的是：

- 精选 A 股股票池
- `AKShare` 历史日线数据
- 本地训练与本地打分

说明：

- 当前环境里直接拉“全市场实时股票列表”可能会受到代理或网络链路影响
- 所以这版优先采用“指定股票池 + 历史日线”的方式先把真实 A 股流程跑通
- 后续可以继续扩展到更大的股票池和更完整的实时模拟盘

## 页面能力

当前网页支持：

- 查看股票池规模、特征数量、训练样本数量
- 查看当前数据来源
- 查看最新候选股排名
- 查看当前模型运行状态
- 点击按钮重新训练模型
- 点击按钮刷新真实 A 股数据

## 独立训练脚本

如果你想单独训练模型，可以执行：

```powershell
cd D:\Projects\codex\Agu
.venv\Scripts\python scripts\train.py
```

脚本会优先尝试：

1. 拉取真实 A 股历史数据
2. 如果失败，则回退到本地样例数据
3. 自动训练并输出模型指标

## API 概览

- `GET /api/health`
  - 健康检查
- `GET /api/overview`
  - 返回模型总览信息
- `GET /api/picks`
  - 返回候选股列表
- `POST /api/train`
  - 重新训练模型
- `POST /api/refresh-real-data`
  - 刷新真实 A 股数据并重新训练

## 当前版本说明

- 当前默认优先使用真实 A 股历史数据
- 当前模型是一个基础版 `LightGBM` 回归排序模型
- 当前页面主要用于打通 A 股选股模型工作流
- 这版还没有接入实时盘口级行情
- 这版还没有接入 `vn.py + PaperAccount` 做本地模拟盘执行

## 后续可扩展方向

1. 扩展股票池到沪深 300、中证 500 或自定义列表
2. 增加更多因子特征，如基本面、行业、风格和成交量特征
